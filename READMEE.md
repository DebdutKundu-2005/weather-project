# Weather App (Python)

A simple Python weather application that fetches real-time weather
information based on city name.

## Features
- City-based weather search
- Real-time temperature display
- Beginner-friendly Python code
- Uses live weather API

## Technologies Used
- Python
- Weather API
- Git & GitHub
